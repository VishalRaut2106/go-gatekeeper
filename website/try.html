Try it instantly

Run a secure session directly from your command line without downloading the desktop application.

Relay Network Active
npx gatekeeper-shell
1

Open your Terminal

Launch Terminal (macOS/Linux) or Command Prompt/PowerShell (Windows).

2

Paste & Run the Command

Copy the command above and press Enter. Node.js (npm) will execute the package temporarily.

3

Start Collaborating

Your session ID will be generated. Send it to your peer to establish an end-to-end encrypted connection.
